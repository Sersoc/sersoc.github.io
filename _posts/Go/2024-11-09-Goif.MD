> Go에서는 조건문이 조금 자유분방한 느낌이 있다. 아래의 예시들을 보면서 살펴보자

## if
구문은 일반적인 if 조건문 {} 식이다. 그런데 Go에서는 특이하게 if Func();조건문 {} 이런식으로 가능하다.
``` go
func ex()(int,int){ // Go에서는 return 값을 여러개를 보내줄 수 있다. 나중에 자세히 후술할 예정이다.
    return 1,2
}
if a,b := ex(); a>b{
    fmt.print(a)
} 
````
이런식으로 조금 색다르게 if문을 구현할 수 있다.
